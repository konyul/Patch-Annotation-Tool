# Patch Annotation Tool


## 사용방법 (Window)


Anaconda 설치

Conda 가상환경 생성
```
conda create --name={가상환경명} python=3
conda activate {가상환경명}
```

디렉토리 생성
```
git clone https://github.com/konyul/Pixel-Annotation-Tool.git
```


환경 설정
```
cd Pixel-Annotation-Tool
pip install -e .
```

명령어 실행 (Pixel-Annotation-Tool 외부 폴더에서 실행)
```
labelme
```
